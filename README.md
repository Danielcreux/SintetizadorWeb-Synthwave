# SintetizadorWeb-Synthwave

## Descripción
Sintetizador web con diseño estético "Synthwave", implementado en un único archivo HTML. Incluye interfaz interactiva con controles maestros y teclado virtual, enfocado en una experiencia visual retrofuturista.

## Características
- **Tema oscuro** con colores base `#0a0a0f`, acentos cian (`#00f0ff`), magenta (`#ff00aa`) y dorado (`#ffd700`).
- **Controles maestros** con deslizadores de rango y botón de inicio con efectos de sombra y escala al pasar el cursor.
- **Teclado virtual** con teclas blancas (60x200px) y negras, estilizadas con sombras 3D y bordes redondeados.
- Logotipo con degradados cian/magenta.

## Uso
1. Abrir el archivo `Sintesis.html` en un navegador web moderno.
2. Interactuar con los controles maestros y el teclado virtual para generar sonidos.
3. Usar el botón de inicio para activar la interfaz (efecto visual al hacer hover).
